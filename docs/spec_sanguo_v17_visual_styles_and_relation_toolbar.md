# 三國演義探險地圖 V17：視覺風格系統與關係人物工具列規格

版本：`v17-visual-style-system-relation-toolbar`

## 目標

V17 將「顯示風格」從細微顯示偏好改為真正的視覺呈現方法。使用者切換風格時，應明顯感覺到整體空間、配色、卡片質感、地圖背景與互動層次不同，而不是只有線條變淡。

同時重構 `關係人物` 的功能列，讓它像閱讀問題，而不是一排技術 filter。

## 風格介面契約

`state.visualStyle` 僅允許三個值：

- `classic`：預設，延續目前馬卡龍清爽風格。
- `bamboo`：竹簡書院，偏閱讀、紙感、暖色、沉穩。
- `warroom`：戰棋沙盤，偏策略、深色、強對比、地圖感。

CSS 對外契約：

- `body.style-bamboo`
- `body.style-warroom`

不再使用 `style-clean` / `style-reading` 作為主要風格名。這避免未來把「功能偏好」誤當成「視覺系統」。

## 字體大小入口

字體大小從設定抽屜移到上方工具列，讓它成為主要閱讀輔助控制。

顯示形式：

- `字` 分段按鈕：`小 / 中 / 大`
- 仍使用 `state.fontScale = small | normal | large`
- 保留 localStorage 記憶

## 新增兩種風格

### 竹簡書院 `bamboo`

用途：適合閱讀人物、章回脈絡與故事證據。

視覺特徵：

- 背景偏宣紙與竹簡：米白、竹青、墨綠、赭石。
- 卡片像紙面分區，邊框柔和，陰影較低。
- 地圖區降低彩度，讓文字和證據更像閱讀資料。
- 連線透明度較低，節點維持陣營色但變得更柔。

### 戰棋沙盤 `warroom`

用途：適合看戰役、路徑與策略關係。

視覺特徵：

- 背景偏深墨藍與沙盤格線。
- 卡片深色、邊線較亮，強化「戰略桌」感。
- 地圖背景對比提高，主線與選取節點更醒目。
- 文字保持高對比，不使用低灰字。

## 關係人物工具列

標題改為單一閱讀問題：

`關係人物`  
`他們是什麼樣的關係呢？`

右側或同一行顯示單選標籤：

- `戰鬥`
- `幫助`
- `說服`
- `主從`
- `受挫`

互動規則：

- 單選。
- 點同一個已選標籤會取消，取消後代表不套用分類，顯示所有重點關係。
- 不需要 `全部` 按鈕。
- 舊的 `全部`、`勝負轉折` 不在人物模式工具列出現。

內部資料仍沿用 `PERSON_RELATION_TABS`，但介面 label 和 filter contract 改成更符合兒童閱讀語言。

## 驗收條件

- HTML 版本為 `v17-visual-style-system-relation-toolbar`。
- 上方工具列可直接切換 `小 / 中 / 大`。
- 設定抽屜只保留整體風格與其他進階設定，不再放字體大小。
- 整體風格至少有 `竹簡` 與 `沙盤` 兩個明顯不同按鈕。
- `關係人物` 顯示副標 `他們是什麼樣的關係呢？`。
- 人物模式 filter 為 `戰鬥 / 幫助 / 說服 / 主從 / 受挫`，可單選並可取消。
- 生成後輸出正式檔 `三國演義探險地圖.html` 與版本檔 `三國演義探險地圖_v17.html`。
